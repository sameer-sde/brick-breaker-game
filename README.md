# Brick Breaker Game

A classic **Brick Breaker** game built using **HTML, CSS, and JavaScript**. This project demonstrates front-end web development skills, game logic implementation, and interactive UI design.

## Features

- Bounce the ball using the paddle to break bricks  
- Score tracking for each brick destroyed  
- Smooth animations and responsive gameplay  
- Works on any modern web browser  

## Tech Stack

- **HTML** – Game structure  
- **CSS** – Styling and layout  
- **JavaScript** – Game logic, collision detection, and interactions  

## How to Play

1. Open `index.html` in a web browser  
2. Use the **left and right arrow keys** to move the paddle  
3. Bounce the ball to break all the bricks and score points  
4. Try to clear all bricks without letting the ball fall
